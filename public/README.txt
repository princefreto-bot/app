========================================
VIDÉOS - EduAnalytics Pro
========================================

Ce dossier contient les vidéos de l'application.

========================================
1. VIDÉO DE FOND (BACKGROUND)
========================================

La vidéo de fond tourne EN BOUCLE derrière toute l'application.
Les animations et le contenu s'affichent PAR-DESSUS.

OPTION A - UNE SEULE VIDÉO :
----------------------------
Placez votre vidéo ici et nommez-la :
  → background.mp4

Elle tournera en boucle automatiquement.


OPTION B - PLUSIEURS VIDÉOS :
-----------------------------
Pour plusieurs vidéos qui se succèdent :
  → video1.mp4
  → video2.mp4
  → video3.mp4
  → etc.

Elles seront lues dans l'ordre, puis recommenceront.


========================================
2. VIDÉO D'INTRODUCTION (OPTIONNEL)
========================================

Pour une vidéo d'intro AU DÉMARRAGE (une seule fois) :
  → intro.mp4

Elle sera jouée avant d'afficher l'application.
Bouton "Passer" disponible.


========================================
SPÉCIFICATIONS RECOMMANDÉES
========================================

Pour BACKGROUND (doit être fluide et léger) :
- Durée : 10-60 secondes (boucle)
- Résolution : 1920x1080 (Full HD)
- Codec : H.264 (MP4)
- Bitrate : 2-5 Mbps
- Poids : < 50 MB
- Style : Abstrait, slow motion, ambient

Pour INTRO (une seule lecture) :
- Durée : 5-15 secondes
- Résolution : 1920x1080
- Poids : < 15 MB

========================================
FORMATS SUPPORTÉS
========================================
✓ MP4 (H.264) - RECOMMANDÉ
✓ WebM (VP9)
✓ MOV (pour Safari)

========================================
IDÉES DE VIDÉOS BACKGROUND
========================================
- Particules abstraites flottantes
- Dégradés de couleurs animés
- Formes géométriques en mouvement
- Ondes lumineuses
- Effet bokeh / lumières floues
- Lignes de code qui défilent
- Réseaux de connexions (nodes)

Sites pour vidéos gratuites :
- pexels.com/videos
- pixabay.com/videos
- coverr.co
- mixkit.co
